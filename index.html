Sobre mí

Soy un desarrollador web freelance con pasión por construir interfaces que sean tanto visualmente atractivas como técnicamente sólidas. Me motiva transformar ideas en productos digitales que las personas disfruten usar.

Mi enfoque se centra en las bases: HTML semántico, CSS moderno y JavaScript vanilla. Creo que dominar los fundamentos es la clave para crear experiencias web que rindan bien, sean accesibles y mantengan un código limpio y mantenible.

Aunque trabajo principalmente con tecnologías del lado del cliente, también tengo experiencia con React para proyectos que lo requieren. Siempre estoy aprendiendo y explorando nuevas herramientas para mejorar mi flujo de trabajo.
